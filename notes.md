NOTES:

  1. Update node.js using nvm
      - install nvm:
        - sudo apt update
        - curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
        - source ~/.bashrc
        - nvm --version
        - nvm ls-remote - lists node versions
        - nvm install [version]
  
  2. Create Next.js app:
      - npx create-next-app
